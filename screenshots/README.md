# Screenshots

This directory contains screenshots for the NYAAY AI README and documentation.

## Planned Screenshots

| Filename | Description |
|----------|-------------|
| `landing.png` | Landing page / hero section |
| `dashboard.png` | User dashboard with chat history |
| `reasoning.png` | Legal Reasoning Engine in action |
| `drafting.png` | Document Drafting interface |
| `upload-chat.png` | Upload & Chat with a PDF document |

## Adding Screenshots

1. Run the application locally
2. Capture each screen at **1280×800** or higher resolution
3. Save as PNG with the filenames above
4. Reference them in `README.md` using:

```markdown
![Legal Reasoning](screenshots/reasoning.png)
```
